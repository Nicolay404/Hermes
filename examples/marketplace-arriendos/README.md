# Marketplace Arriendos — Placeholder

Este ejemplo está reservado para un marketplace de arriendos estudiantiles.

Para inicializarlo con el sistema completo, usa:

```bash
cd ai-project-operating-system
bash scripts/setup.sh marketplace-arriendos
```

Y luego completa los documentos siguiendo la metodología de `docs/15_AI_WORKFLOW.md` y el prompt de descubrimiento `prompts/01_project_discovery.md`.

---

**Idea base:** Plataforma para conectar estudiantes universitarios con arrendatarios cerca de las universidades en Ecuador. Problema validado: los estudiantes buscan arriendos por grupos de Facebook sin garantías, fotos reales ni proceso seguro.
