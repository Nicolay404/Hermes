## Cómo usar este documento

1. Crea un nuevo CR usando `templates/change_request_template.md`
2. Asigna el próximo ID disponible (CR-XXX)
3. Pega el CR al final de este archivo
4. Actualiza el índice
5. Cambia el estado conforme avanza

**Estados válidos:**
- `Pendiente` — propuesto, no analizado
- `En análisis` — se está evaluando el impacto
- `Aprobado` — autorizado para implementar
- `En implementación` — se está desarrollando
- `En revisión` — QA / code review
- `Completado` — implementado y documentado
- `Rechazado` — descartado con justificación
