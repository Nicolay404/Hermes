# Tareas Completadas

> Historial de todo lo que se ha completado. Útil para retrospectivas y para que cualquier IA entienda el progreso del proyecto.

---

## Cómo usar este archivo

Mover aquí las tareas del `active_sprint.md` cuando se completan. Incluir siempre la fecha y una nota breve de qué se implementó.

---

## Formato

```
### [YYYY-MM-DD] — Sprint [N]

| ID | Tarea | CR asociado | Notas |
|---|---|---|---|
| T-XXX | [Descripción] | CR-XXX | [Qué se hizo / decisiones tomadas] |
```
