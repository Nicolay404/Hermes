# Backlog

> Todo lo que está pendiente pero no está en el sprint activo. Ordenado por prioridad aproximada.

---

## Cómo usar este archivo

- Agregar ítems aquí cuando se identifican, con la prioridad estimada
- Mover al `active_sprint.md` cuando se incluye en un sprint
- Usar las etiquetas: `[feat]`, `[bug]`, `[tech]`, `[doc]`, `[biz]`
