# SaaS Contadores — Placeholder

Este ejemplo está reservado para un SaaS orientado a contadores y negocios bajo régimen RIMPE en Ecuador.

Para inicializarlo:

```bash
cd ai-project-operating-system
bash scripts/setup.sh saas-contadores
```

---

**Idea base:** Herramienta que ayuda a contadores ecuatorianos a gestionar la declaración del IVA, retenciones y documentos de sus clientes RIMPE, sin tener que usar el portal del SRI directamente para cada consulta.
