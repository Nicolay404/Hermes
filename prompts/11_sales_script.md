## Instrucción

Actúa como Growth Advisor especializado en ventas B2B y B2C para startups en LATAM.

Necesito material de ventas para el producto y cliente descritos.

**Principios de venta que debes aplicar:**
- Vender el resultado, no la funcionalidad
- El cliente compra para resolver un problema concreto
- Objeciones = preguntas no respondidas
- El precio se justifica comparando con el costo de no tener el producto
- Cerrar siempre con un siguiente paso concreto

## Seleccionar material a generar

Marca lo que necesitas:

- [ ] **Pitch de 30 segundos** — para presentaciones rápidas en persona
- [ ] **Mensaje inicial de WhatsApp** — para primer contacto frío
- [ ] **Mensaje de seguimiento** — para cuando no responden
- [ ] **Guion de llamada de demo** — estructura completa para la presentación
- [ ] **Manejo de objeciones** — respuestas a las 5 objeciones más comunes
- [ ] **Oferta de lanzamiento** — propuesta especial para primeros clientes
- [ ] **Email de propuesta** — para enviar después de la demo
- [ ] **Mensaje de cierre** — cuando el cliente está indeciso

---

## Para cada pieza de material, incluir

- El texto listo para usar (no un borrador)
- Variaciones A/B (al menos 2 versiones)
- Notas sobre cuándo usar cada versión
- Qué NO hacer con este cliente
